# Brand Guardian AI - Interactive Prototype

## 🎯 What This Is

A production-quality React prototype demonstrating **Brand Guardian AI** - an AI-powered system that prevents brand drift by detecting deprecated assets in real-time and suggesting current versions before approval.

## 🚀 Live Demo

**Link:** [To be deployed]

## 💻 Running Locally

```bash
# Install dependencies
npm install

# Start dev server
npm run dev

# Build for production
npm run build
```

## 🎨 Features Demonstrated

1. **Real-Time Asset Detection**
   - Drag-and-drop brand assets onto canvas
   - AI instantly detects deprecated versions

2. **Visual Diff Warning**
   - Side-by-side comparison of old vs. current assets
   - Highlights specific changes (tagline, color, icon style)
   - Non-intrusive tooltip interface

3. **One-Click Correction**
   - "Switch to Current Version" button
   - Smooth asset swap animation
   - Success feedback

4. **Brand Compliance Tracking**
   - Real-time compliance indicator (0% → 100%)
   - Submit button enables only at 100% compliance

## 🛠️ Tech Stack

- **React 18** + **TypeScript**
- **Tailwind CSS** (Canva design system)
- **Framer Motion** (smooth animations)
